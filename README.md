# Spencffirmations

A simple web app that delivers positive life affirmations with a friendly bro twist!

## Features

- Click to get a new random affirmation
- Each affirmation ends with a casual bro suffix (e.g., "my guy", "bro", "king")
- Clean, modern UI with smooth animations
- Fully responsive design
- Works on any device

## How to Use

1. Open `index.html` in your web browser
2. Click the button or the affirmation box to get a new positive message
3. Repeat whenever you need a boost!

## Project Structure

- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - Affirmation logic and interactivity

## Customization

You can easily add more:
- **Affirmations**: Edit the `affirmations` array in `script.js`
- **Bro Suffixes**: Edit the `broSuffixes` array in `script.js`
- **Colors**: Modify the gradient and colors in `style.css`
